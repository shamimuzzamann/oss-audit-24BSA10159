## About This Project
This repository contains five shell scripts written as part of the Open Source Audit capstone project. The scripts demonstrate practical Linux and Bash scripting skills, all themed around the Linux Kernel — the open-source operating system core originally written by Linus Torvalds in 1991 and released under the GNU General Public License v2.

## How to Run Each Script on Linux

### Step 1 — Clone the repository
```bash
git clone https://github.com/shamimuzzamann/oss-audit-24BSA10159.git
cd oss-audit-24SA10159/Scripts
```

### Step 2 — Make scripts executable
```bash
chmod +x *.sh
```

### Script 1 — System Identity Report
```bash
./script1_system_identity.sh
```
Displays your Linux distribution, kernel version, architecture, username, home directory, uptime, date/time, and the GPL v2 license message.

### Script 2 — FOSS Package Inspector
```bash
./script2_package_inspector.sh
```
Checks whether the Linux kernel image package is installed via `dpkg`, shows version details, and uses a `case` statement to print a one-line philosophy note about the software.

### Script 3 — Disk and Permission Auditor
```bash
./script3_disk_auditor.sh
```
Loops through `/etc`, `/var/log`, `/home`, `/usr/bin`, `/tmp`, `/boot`, `/proc`, `/sys` and prints permissions, owner, and size. Also audits kernel-specific directories (`/proc`, `/sys`, `/lib/modules`).

### Script 4 — Log File Analyzer
```bash
./script4_log_analyzer.sh /var/log/dpkg.log install
```
- First argument: path to any log file
- Second argument: keyword to search for (default: `error`)
- Counts occurrences and prints the last 5 matching lines.

Other examples:
```bash
./script4_log_analyzer.sh /var/log/auth.log sudo
./script4_log_analyzer.sh /var/log/dpkg.log upgrade
```

### Script 5 — Open Source Manifesto Generator
```bash
./script5_manifesto.sh
```
Asks you 3 interactive questions and generates a personalised open source philosophy statement, saved to `manifesto_<username>.txt`.
